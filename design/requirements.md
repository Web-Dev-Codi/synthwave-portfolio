# Synthwave Portfolio — Section Requirements

---

## 🌐 Global / Sitewide

- Deep navy/black base background color
- Starfield that persists across all sections (slow twinkle animation)
- Neon glow on all interactive elements (links, buttons, hover states)
- Consistent color palette: hot pink, cyan, purple, deep navy
- Pixel/chrome gradient font for all headings
- Scanline overlay with a subtle purple tint across the whole page
- Smooth glitch transition effect between sections on scroll
- Neon horizontal rule dividers between every section
- Subtle CRT vignette on the screen edges

---

## 1. 🎬 Hero / Landing Section

- Full viewport height
- Animated synthwave grid floor scrolling toward the viewer
- Retrowave striped sun rising from the horizon, centered
- Wireframe mountains on the left and right flanking the sun
- Your name as the headline in large pixel/chrome gradient text
- Your title (e.g. "Full Stack Developer") below in glowing cyan
- Blinking cursor at the end of your title
- A glowing CTA button ("View My Work") with neon border and bloom on hover
- Subtle VHS noise/grain layer on top of the whole hero

---

## 2. 👾 About Me Section

- Dark card or panel with a faint inner grid texture
- A pixel art avatar or portrait of yourself (instead of a photo)
- Short bio text in a clean sans-serif (not pixel font — save that for headings)
- Neon-glowing section heading ("ABOUT ME" in chrome gradient)
- A few key facts displayed as glowing stat blocks — e.g. years of experience, location, current focus
- Subtle animated glow pulse on the avatar or card border

---

## 3. 🛠️ Skills Section

- Section heading in pixel chrome style
- Skills grouped by category (Frontend, Backend, Tools, etc.)
- Each skill displayed as a **neon badge** or retro chip with glow
- Skill bars or level indicators styled as retro arcade progress bars (blocky, pixelated fill)
- Color coding by category — e.g. frontend in cyan, backend in pink, tools in purple
- Hover on each badge causes a bloom/brighten effect

---

## 4. 🗂️ Projects Section

- Section heading in pixel chrome style
- Project cards styled as dark panels with glowing neon borders (alternating pink and cyan)
- Faint grid texture inside each card
- Project screenshot or a synthwave-themed placeholder illustration per card
- Project title in gradient pixel font
- Short description in readable sans-serif
- Tech stack shown as small glowing retro tags at the bottom of each card
- Two glowing CTA buttons per card: "Live Demo" and "GitHub" with distinct neon colors
- Card hover effect: border glow intensifies, slight upward lift

---

## 5. 💼 Experience / Timeline Section

- Vertical timeline styled as a glowing neon line running down the center
- Each entry is a dark card positioned left/right alternating
- Company name in gradient text, role in cyan, dates in muted purple
- Timeline nodes (the dot on the line) styled as glowing neon circles
- Subtle entrance animation as each entry scrolls into view

---

## 6. 📬 Contact Section

- Section heading in pixel chrome style
- Short punchy tagline ("Let's Build Something" etc.) in neon pink
- Input fields styled with dark backgrounds, neon cyan borders, and glow on focus
- Labels above inputs in muted purple
- Submit button large, glowing, with a bloom animation on hover/click
- Social links (GitHub, LinkedIn, etc.) as neon icon buttons
- Optionally: a small retro synthwave sun or grid decorative element behind the form

---

## 7. 🔻 Footer

- Minimal — dark background, centered
- Your name or logo in small pixel font
- Copyright line in muted purple
- Thin neon line at the very top of the footer as a divider
- Social icon links repeated here with subtle glow

---

## 🎨 Assets Needed

- Pixel art avatar / portrait
- Synthwave sun graphic (can be CSS or SVG)
- Wireframe mountain graphics (SVG or canvas)
- Scrolling grid animation (CSS or canvas)
- Chrome/gradient pixel font (e.g. Press Start 2P + CSS gradient clip)
- Neon glow CSS utility classes applied globally
- Optional: a looping synthwave background track with a mute toggle
